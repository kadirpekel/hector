**Pure A2A-Native Declarative AI Agent Platform**

Build powerful AI agents in pure YAML. Compose single agents, orchestrate multi-agent systems, and integrate external A2A agents—all through declarative configuration and industry-standard protocols.

## 🌟 Why Hector?

<div class="features-grid">
  <div class="feature">
    <h4>🎯 Declarative First</h4>
    <p>Define complete agent systems in pure YAML. No code required.</p>
  </div>
  
  <div class="feature">
    <h4>🔗 A2A Native</h4>
    <p>Built on open Agent-to-Agent protocol for true interoperability.</p>
  </div>
  
  <div class="feature">
    <h4>🚀 Multi-Agent Ready</h4>
    <p>LLM-driven orchestration with native and external agents.</p>
  </div>
  
  <div class="feature">
    <h4>🔧 Extensible</h4>
    <p>gRPC plugin system for custom LLMs, databases, and tools.</p>
  </div>
  
  <div class="feature">
    <h4>🏭 Production Ready</h4>
    <p>Authentication, streaming, sessions, monitoring built-in.</p>
  </div>
  
  <div class="feature">
    <h4>⚡ Developer Friendly</h4>
    <p>Single binary, zero dependencies, simple CLI.</p>
  </div>
</div>
